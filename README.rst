ansible-role-cachet-docker
==========================
Installs Cachet_ using it's provided `Docker-compose repository`_ on a CentOS
host.

.. _Cachet: https://cachethq.io/
.. _Docker-compose repository: https://github.com/CachetHQ/Docker
